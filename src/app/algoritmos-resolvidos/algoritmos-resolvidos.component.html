Definição de Algoritmo:

Conjunto das regras e procedimentos lógicos perfeitamente definidos que levam à solução de um problema em um número finito de etapas.

Representação em Pseudocódigo:

                    
    inicio
        inteiro n1, n2, result      // declaração variável

        ler(n1)                     // leitura do numero 1
        ler(n2)                     // leitura do numero 2
        
        result <- n1 + n2           // somar e atribuir 
                                    // à variável result
    
        escrever(result)            // imprimir o resultado
    fim
